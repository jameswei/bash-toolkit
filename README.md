Bash ToolKit

## Overview

configuraion and tools for developers in Bash for efficiency.

## Installation

#. method 1

* $ git clone ...
* $ cd ... && make install
* append the follow code snippets into ~/.bashrc or "~/.bash_profile"

```bash
if [ -f ~/.bash.d/bashrc ]; then
    . ~/.bash.d/bashrc
fi
```

# method 2

* copy .bash.d and .local into ~/
* append the follow code snippets into ~/.bashrc or "~/.bash_profile"

```bash
if [ -f ~/.bash.d/bashrc ]; then
    . ~/.bash.d/bashrc
fi
```
